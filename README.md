# dotfiles
dot files repo

# install instructions
  1. Clone the git repo to $HOME

  2. Run ./dotfiles/install.sh OR
     selectively source the startup scripts from $HOME/dotfiles as needed.

     Check out *./dotfiles/install.sh --help* for all options
     *./dotfiles/install.sh --dryrun* => to check out dryrun mode before overwriting any startup files
